# 📊 IT Expenditure Variance Analysis

## 🚀 Project Overview
This project analyzes monthly IT expenditure data to evaluate budget performance, identify overspending trends, and measure forecast accuracy.

The objective is to transform raw financial data into actionable business insights using Python-based data analysis and visualization techniques.

---

## 🎯 Business Objective

Organizations must continuously monitor IT spending to avoid budget overruns and improve financial planning.

This analysis answers key business questions:

- 📌 Which month had the highest overspending?
- 📌 What is the average variance percentage?
- 📌 How accurate were the forecasts compared to actual spending?
- 📌 Is IT spending increasing over time?

---

## 📂 Dataset Description

The dataset contains monthly IT financial information including:

- Month
- Planned Budget
- Actual Spending
- Forecast Spending

---

## 📊 Key Metrics Calculated

### 1️⃣ Variance
Variance = Actual Spending − Planned Budget

### 2️⃣ Variance Percentage
Variance % = (Variance / Planned Budget) × 100

### 3️⃣ Forecast Deviation
Forecast Deviation = Actual Spending − Forecast
